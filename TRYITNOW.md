# 试用 AI 首席参谋（5 步开始）

> 你拿到了 AI 首席参谋的下载链接，这份文档告诉你接下来该怎么做。
> 全程约 **20-30 分钟**（其中大部分时间在等机器自己装东西，你可以去倒杯水）。

---

## 你需要准备

| 项目 | 说明 |
|------|------|
| 一台 Windows 10/11 电脑 | 内存 ≥ 8GB，硬盘空闲 ≥ 30GB |
| 一根能上网的网线/Wi-Fi | 用来下载 Docker、AI 模型、依赖 |
| 一个手机号 | 去 SiliconFlow 注册（免费，只需一次） |

---

## 第 1 步：下载安装包

打开浏览器，访问：

**https://github.com/AI-For-Management/ai-chief-of-staff/releases/latest**

找到 **Assets** 区域，点击 `AI首席参谋-Setup-0.1.0.exe`（约 11 MB）下载。

> 国内访问 GitHub 偶尔慢，挂代理或换个时间段重试即可。

---

## 第 2 步：装 Docker Desktop（电脑只装一次）

> AI 首席参谋的所有服务跑在 Docker 里。这是装一次就忘记的事。

1. 打开 https://www.docker.com/products/docker-desktop/
2. 点蓝色 **"Download Docker Desktop"** → 选 Windows 版下载
3. 双击下载下来的 `Docker Desktop Installer.exe`
4. 一路点 **Next** / **Continue** / **Install**，最后 **重启电脑**
5. 重启后从开始菜单打开 **Docker Desktop**
6. 第一次启动会让你登录或同意协议——**点 "Skip"（跳过）即可**
7. 等右下角任务栏的鲸鱼图标从橙色变成白色（约 30 秒），说明 Docker 已就绪

---

## 第 3 步：申请 AI API Key

> AI 首席参谋用国内的 SiliconFlow 大模型服务。注册免费、新用户送 14 元额度，足够试用一阵。

1. 打开 https://siliconflow.cn → 右上角 **"登录/注册"**
2. 用手机号注册（也可微信扫码）
3. 进入控制台 → 左侧 **"API 密钥"** → **"新建 API 密钥"**
4. 起个名（如 "ai-secretary"）→ **"新建"**
5. 弹出的字符串（`sk-xxxx...`）**立刻复制保存**——关掉就再也看不到了

---

## 第 4 步：双击安装包

1. 双击第 1 步下载的 `AI首席参谋-Setup-0.1.0.exe`
2. **Windows 可能弹蓝色警告**（"Windows 已保护你的电脑"），因为这是未签名的开发版
   - 点 **"更多信息"** → **"仍要运行"**
3. 标准 Windows 安装向导（中文）一路 **下一步** 即可
4. 最后桌面会出现「AI 首席参谋」图标

---

## 第 5 步：第一次启动

1. 双击桌面图标
2. 弹出**配置向导**：
   - **SiliconFlow API Key**：粘贴第 3 步的 `sk-xxx...`
   - **管理员密码**：自己想一个（至少 12 位、带大小写和数字），**记好**——这是你以后登录的密码
   - 点 **"保存配置并启动"**
3. 关掉配置窗口后，**等 1-3 分钟**——后台在拉取 Docker 镜像、启动 6 个服务
4. 看到右下角弹通知 **"AI 首席参谋已启动"** + 浏览器自动打开 → 进入登录页
5. 登录：
   - 用户名：`admin`
   - 密码：第 5.2 步你自己设的那个
6. 看到「综合面板」就成功了 🎉

---

## 之后每次怎么用

- **开机后**：双击桌面图标 → 等 1 分钟 → 浏览器自动打开
- **不用时**：浏览器关掉就行；想完全停掉服务，右键托盘鲸鱼/Logo → "停止服务"
- **退出启动器**：右键托盘图标 → "退出启动器"（服务仍在后台跑）

## 常见问题

### Q：双击 .exe 弹"Windows 已保护你的电脑"
正常的，因为没买代码签名证书。点 **更多信息 → 仍要运行**。

### Q：配置完后等了 3 分钟浏览器还没自动开
- 检查 Docker Desktop 鲸鱼图标是不是白色（如果是橙色还在启动中，再等等）
- 手动打开浏览器访问 http://localhost:8501
- 还不行 → 右键托盘 AI 图标 → **查看日志**，看里面有没有 ERROR

### Q：AI 一直转圈不回答
通常是 SiliconFlow API key 没填对或额度用完了。回到第 3 步重新拿 key，编辑 `C:\Program Files\AI 首席参谋\.env` 修改 `SILICONFLOW_API_KEY` 那一行，然后右键托盘图标 → **重启服务**。

### Q：想彻底删掉
开始菜单 → 搜 "卸载 AI 首席参谋" → 双击。或者控制面板「添加或删除程序」找到它点卸载。

### Q：想看完整使用说明
- 用户视角详细说明：[`docs/INSTALL.md`](docs/INSTALL.md)
- 运维：[`docs/OPERATIONS.md`](docs/OPERATIONS.md)
